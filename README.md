IMPACT Interoperability Framework
=================================

The IMPACT Interoperability Framework (IIF) specifies the interoperability 
layer of software components and content developed in the IMPACT project.

Components:
----------- 

* toolwrapper
* interfaces
    * taverna
	* t2-client
	* t2-webdav
	* _t2-plugin_
    * web
        * wsdl-client
